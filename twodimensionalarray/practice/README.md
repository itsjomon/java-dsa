## Qn 3.

Write a program to Find **Transpose** of a Matrix.

What is Transpose?

Transpose of a matrix is the process of **swapping the rows to columns**. For a 2x3 matrix, 

- <mark>Matrix</mark>

    a11  a12  a13

    a21  a22  a23

- <mark>Transposed Matrix</mark>

    a11  a21

    a12  a22

    a13  a23
