# Practice Questions

## Qn 1.

What is the value of `x^x` for any value of `x`?

**Ans:** The value of `x^x = 0`, XOR gives `0` when the bits are same. If we compare the same number to itself, the bits will always be the same. So, the answer of `x^x` will always be `0`
